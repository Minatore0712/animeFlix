animeFLix

A web application that provides users with access to information about different Anime movies, directors and genres. Users are able to register, update their personal information and create a list of their favorite movies.

Request URL HTTP Method URL parameters Request body data format Respond body data format
Get a list of ALL movies /movies GET None A JSON object holding data about all movies
Get data about a single movie by title /movies/[Title] GET /movies/:Title None A JSON object holding data about a single movie containing a title, description, genre, director, image and featured.

Example:
{
"title": "Violet Evergarden",
"description": "Violet Evergarden, the child soldier turned Auto Memory Doll, writes letters that evoke the words her clients can't. But when a terminally ill boy requests her services for her family, her own feelings about love and loss resurface.",

"director": "Taichi Ishidate",
"genre":"Drama",
"image": "violet.png",
"featured": true
}
Get data about a movie genre by name /movies/genres/[Genre] GET /movies/genres/:Genre None A JSON object holding data about the genre
Get data about a movie director by name /movies/directors/[Name] GET /movies/directors/:Name None A JSON object holding data about the director
Register new user /users POST A JSON object holding data about the user to create.
Example:
{
ID: Integer,
Username: String (required),
Password: String (required),
Email: String (required),
Birthday: Date
} 		A JSON object holding data about the user that was added.
Example:

"_id": "607467938f1de1634b2326d4",
"Username": "Maria",
"Password": "password",
"Email": "maria@web.de",
"Birthday": "1992-08-02"
}
Update a user information /users/[Username] PUT /users/:username a JSON object holding data about the user to update.
Example:
charset="{ "Username": "Maria",

Add a movie to Favourites /users/[Username]/favourites/[MovieID] POST /users/:username/favourites/:MovieID None A JSON object holding data about the movie that was added to the favourites Movie list.
Example:
{ “title” : “Violet Evergarden"}
Remove movie from Favourites /users/[Username]/favourites/[MovieID] DELETE /users/:username/favourites/:MovieID None A JSON object holding data about the updated favorite Movies list.
Unregister an existing user /users/[Username] DELETE /users/:username None A text message indicating whether the user was successfully deleted
